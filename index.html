Shopping Cart

Project Details

My initial learning goal for JavaScript was to build a client-side shopping cart to handle User Interaction. This vanilla JS solution uses browser localStorage capabilities to hold a JSON object that can be structured and delivered to a server as necessary, such as the paypal API, which would then handle security-critical information. This example is a simplified version of the integration, which is fully developed, tested with the PayPal sandbox, and ready to deploy.

Understanding localStorage functionality has lead to other useful solutions, such as tracking pagevisits to trigger or suppress modal communication events.

If you are interested in examining the localStorage state, this button will log it to the developer console.

A Note on Math - This cart uses a best-practice solution to JS floating point limitations by scaling the financial calculation to cents, allowing for tallying in whole integers.

JavaScript Tricks

Inversion Three Ways

Sometimes overcoming problems requires flexibility. The ACME Brand Word Encrypter takes this very literally, offering three unique solutions to the age old dilemma: how to invert a string.

Array Methods employs built-in JS methods .split(), .reverse(), and .join() to transform the string into an array and then back again.

Decrementing Loop progresses sequentially through the given string, saving each place value to another scoped output variable without changing data type.

Recursion uses self-reference to dig to the bottom of the input using JS string methods .substring() and .charAt() to trigger multiple return values in the stack.

This task teaches flexibility in approach as well as nuance of process. A Functional Programing context would prohibit approaches that mutate data, for example, while program runtime might make one solution preferable for a given data set.

Example Site Design: Uncle Salty's Restaurant

Here is an example page design for a casual counter service restaurant currently under development in Crystal Lake, IL based on a laid-back 'beach bum' persona named Uncle Salty. Mobile-first design was critical as most users will be ordering from their phone.


Value of using Bootstrap front end framework:

- It greatly facilitates mobile-first development by standardizing breakpoints.
- Using class-based styling implies systematic vs. idiosyncratic css.
- The developed nature of the framework means browser compatibility and accessibility principles are ready to go out of the box, saving development time and effort.
